How it translates: each hue is captured as a base (the vivid card color) and a deep tonal (the shape + text on it) — that's the bold poster style up top. The same base then derives a fill (low-opacity tint over the surface), an accent (the bar), and title / body text — the muted FileCard style below. Light and dark each hold their own values under one set of role names, so a card names a hue once and every context paints itself. Toggle to confirm all seven survive the flip.
